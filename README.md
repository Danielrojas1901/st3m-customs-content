# ST3M Customs Content

Plugin personalizado de WordPress para gestionar contenido dinámico reutilizable mediante Custom Post Types, shortcodes, componentes visuales y layouts responsivos.

## Funcionalidades

- Custom Post Type: Sedes
- Custom Post Type: Aliados
- Campos personalizados sin ACF
- Shortcode `[st3m_sedes]`
- Shortcode `[st3m_aliados]`
- Componente reutilizable de cards
- Layout horizontal para sedes
- Layout compacto para aliados
- Placeholder de imagen
- Estilos responsive

## Instalación

1. Descargar el plugin.
2. Subirlo a `wp-content/plugins/`.
3. Activarlo desde WordPress.
4. Usar los shortcodes en una página.

## Shortcodes

```txt
[st3m_sedes]
[st3m_aliados]
